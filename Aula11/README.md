## Testes funcionais usando Selenium

Backend e Frontend de exemplo para testes usando o Selenium.

### Prepação do script de teste

Inicie o backend:
```
cd backend
mvn spring-boot:run
```

Inicie o frontend:
```
cd frontend
pnpm i
pnpm start
```

Abra o frontend no Firefox: http://localhost:3000/

Inicie o plugin do selenium: https://addons.mozilla.org/en-US/firefox/addon/selenium-ide/
